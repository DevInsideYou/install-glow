# Installs [Glow 1.4.1](https://github.com/charmbracelet/glow) on [Ubuntu](https://www.ubuntu.com/)

```bash
curl -fLOs https://script.install.devinsideyou.com/glow
sudo chmod +x glow && ./glow
```
